## Overview

This Next.js starter template provides a solid foundation for building modern, performant, and SEO-optimized personal portfolios and blogs. Built with best practices in mind, it offers a great developer experience while ensuring excellent user experience.

## Key Features

- **Modern Stack**: Next.js 14, React 18, TypeScript
- **Content Management**: MDX support for rich content
- **Styling**: CSS Modules with SCSS support
- **Performance**: Optimized images and fonts
- **SEO**: Built-in metadata and schema support
- **Accessibility**: WCAG 2.1 compliant
- **Dark Mode**: System preference detection
- **Responsive**: Mobile-first design

## Getting Started

1. Clone the repository
2. Install dependencies
3. Start the development server

```bash
git clone https://github.com/yourusername/nextjs-starter-template.git
cd nextjs-starter-template
npm install
npm run dev
```

## Customization

The template is designed to be easily customizable:

- Update content in `src/app/resources/content.ts`
- Modify styles in `src/once-ui/tokens`
- Add new components in `src/once-ui/components`
- Create new pages in `src/app`

## Best Practices

This template follows several best practices:

- **Type Safety**: Full TypeScript support
- **Code Splitting**: Automatic code splitting
- **Image Optimization**: Next.js Image component
- **SEO**: Metadata and schema.org support
- **Performance**: Lighthouse score > 90
